Introduction¶

Software used in this training materials¶

  • GRASS GIS version 8 for geospatial raster, imagery, and vector data processing and providing spatial analysis.

  • GDAL library for reading and writing various geospatial data.

  • sentinelsat library utility to search and download Copernicus Sentinel satellite images.

Sample data¶

For purpose of this training materials a sample data including GRASS 8 binaries (690MB) has been created based on Open Data covering ODSE pilot tiles.
